1.SQL

SQL，structure query language，结构化查询语言。用于对数据库的各种操作。(SQL对大小写不敏感)

SQL分为DML，DDL,DCL三部分

DML，data manipulation language，数据操作语言。
其中包括：
1.SELECT 执行关系查询操作(选择，连接，合并等)
2.UPDATE 修改表中记录的值
3.DELECT 删除表中指定的记录
4.INSERT INTO 向表中插入新的记录
DDL，data definition language，数据定义语言。
其中包括:
1.CREATE 定义新表，视图，宏，索引，触发器和存储过程
2.DROP 删除新表，视图，宏，索引，触发器和存储过程
3.ALTER 修改
DCL, data control language，数据控制语言。
1.CRANT 给用户授予某种权限
2.REVOKE 删除用户某种权限
3.CIVE 转移用户所有权关系
